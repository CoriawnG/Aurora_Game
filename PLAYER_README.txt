Aurora
======

Thanks for playing.

How to start:
- Unzip the whole folder before launching the game.
- Run Aurora.exe.
- If Windows SmartScreen appears, choose "More info" and then "Run anyway" if you trust the file source.

Notes:
- No Python installation is required for the packaged Windows build.
- Music and sound effects are optional and already handled by the build.
- Ending summaries are saved to %LOCALAPPDATA%\Aurora\endings.

If the game does not open:
- Make sure you launched it from the unzipped folder, not from inside the zip viewer.
- Try right-clicking Aurora.exe and choosing Run as administrator only if your system policy blocks unknown apps.
